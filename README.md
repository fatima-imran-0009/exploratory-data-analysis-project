# Exploratory Data Analysis Project

This project performs exploratory data analysis on a dataset using Python, Tableau, Excel, and Power BI.

## Tools Used
- Python
- Tableau
- Excel
- Power BI

## Objective
The goal is to explore, analyze, and visualize the data to discover patterns, trends, and insights that can help in decision-making.

## Description
The project includes:
- Data cleaning and preprocessing using Python and Excel
- Data visualization using Python, Tableau, and Power BI
- Insights and trends discovered from the dataset
